# Ecosystem & Protocol

**Salatto is a LottoFi platform**

> _The infrastructure layer connecting lotteries with decentralized finance._

* Core: **LottoFi (new vertical)**
* By nature: **DeFi-native**
* By UX: **Game-like**
* By infra: **runs on L2 (on Solana now)**



Salatto bridges traditional lottery mechanics with DeFi primitives.

* The lottery ceases to be a “one-way bet.”
* On-chain fairness (VRF)
* Liquidity (ticket buyback)
* DeFi mechanics instead of “dead” tickets

For creators / lottery operators

* No-code lottery creation
* Instant liquidity
* DeFi tools (pools, staking, treasury)
* Global distribution



Live Prize Pools: View participants and funds in real-time.

Verifiable Draws: Provably fair outcomes using on-chain VRF (Verifiable Random Function) technology.

Техническое «мясо» проекта. Если у вас DeFi, GameFi или L2, опишите механику.

* How it works: Архитектура протокола.

* Security & Audits: Ссылки на отчеты об аудитах (CertiK, Hacken и др.). Это критически важно для доверия.
