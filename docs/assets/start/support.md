#### Reporting Issues

If you encounter a bug, failed transaction, or unexpected behavior:

1. Note the game ID and your wallet address
2. Take a screenshot of the issue if possible
3. Describe what happened and what you expected
4. Send the details to [support@salat.to](mailto:support@salat.to)

#### Security Vulnerabilities

If you discover a security vulnerability in Salatto's platform or smart contracts, please report it responsibly:

* Email: [support@salat.to](mailto:support@salat.to) (subject: Security Vulnerability)
* Do not publicly disclose the vulnerability before it has been addressed
* Bug bounty program details will be announced soon
